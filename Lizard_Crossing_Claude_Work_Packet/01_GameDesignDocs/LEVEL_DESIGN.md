# LEVEL DESIGN DIRECTION

## Game structure
Level-based, not open world.

## Level layout
Each level is a sequence of crossing lanes.

The lizard moves forward toward the safe zone.
Hazards move sideways across the lizard's path.

## Lane examples
- pedestrian shoe lane
- stroller wheel lane
- scooter wheel lane
- car/tire lane
- puddle slow lane
- bird shadow lane
- falling leaf/debris lane
- patio chair leg lane

## Early level examples
1. Garden Escape - slow shoes, simple safe path.
2. Sidewalk Shuffle - more pedestrians, faster timing.
3. Boardwalk Rush - scooter wheels and tourists.
4. Curb Gauntlet - car tires, curb drop, storm drain.
5. Patio Panic - sandals, chair legs, crumbs.
6. Midnight Dash - rain, puddles, boots, reduced visibility.

## Difficulty variables
- number of lanes
- hazard speed
- hazard size
- hazard frequency
- collectible risk
- warning time
- level length
- safe path width
- environmental modifier
