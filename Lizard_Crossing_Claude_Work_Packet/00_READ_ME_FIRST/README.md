# Lizard Crossing - Claude Work Packet

This folder is meant to be dropped into the root of the Unity project so Claude Code/Fable can use it as permanent context.

## How to use this folder

1. Put this folder inside your Unity project root.
2. In Claude Code, tell it:
   "Read 00_READ_ME_FIRST/CLAUDE_INSTRUCTIONS.md before doing any work."
3. Do not repeat the full game idea every time.
4. Give Claude one small task at a time.
5. Make Claude run the self-review and testing checklist after each change.

## Core rule

Do not let Claude build random features before the Phase 1 vertical slice feels good.

Phase 1 must prove:
- tiny lizard POV
- giant sideways-moving hazards
- level-based crossing
- readable safe zone
- fun dodge/collect/finish loop
- no console errors
- clean expandable code
